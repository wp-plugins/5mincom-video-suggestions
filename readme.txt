Suggests videos for your post, based on 5min semantic engine.

== Description ==

You just finished writing a great post and you would like to add a video to it ?
Hit the video suggest button and instantly grab a video from 5min.com great video library!

== Installation ==


1. Upload the folders contents to the `/wp-content/plugins/` directory
2. Make sure you have write permissions on your wordpress upload folder
3. Activate the plugin through the 'Plugins' menu in WordPress
